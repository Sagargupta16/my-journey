# Client

React frontend built with [Vite](https://vite.dev/).

## Available Scripts

In the client directory, you can run:

### `npm run dev` (or `npm start`)

Runs the app in development mode on [http://localhost:3000](http://localhost:3000). The page reloads when you make changes.

### `npm run build`

Builds the app for production to the `build` folder.

### `npm run preview`

Serves the production build locally for a final check before deploying.

## Learn More

To learn React, check out the [React documentation](https://react.dev/). For build tooling, see the [Vite guide](https://vite.dev/guide/).
